# Математическа для первого класса

## Тренировка навыков решения примеров для первого класса и подготовки к школе

Приложение генерирует случайные примеры и проверяет правильность их решения. По результатом пройденного урока формируется статистика.
Есть возможность выбрать:

- Максимальное число в примерах
- Тип примера (+, -, +-)

Решать примеры можно в одном из режимов:

- Примеры (ввод ответа с помощью удобной цифровой клавиатуры)
- Тесты (выбор ответа из четырех вариантов)
- Соедини пару (перетаскивание примера на правильный ответ)
  В режиме "Примеры" и "Тесты" доступны подсказки помогающие ребенку понять как нужно решать текущий пример.
